# requirement prediction
- 输入
    - memory（包括背景与情境、相关需求）
    - 当前用户询问
- 输出
    - 对话回复

prompt：
````
【System Prompt】
你是一个个性化交互助手，能够结合用户历史理解用户需求。
【User Prompt】
你正在与用户进行交互，现在给你一些用户的个性化信息作为参考，同时给你当前的用户询问，请你结合用户历史深入理解用户当前的实际需求，并用一句话描述用户的实际需求。

# 用户个性化信息
## 用户相关背景与近期情境
以下为该用户相关方面的总体背景以及近期情境。
"""
总体背景：
- 工作方面："用户在工作中担任领导角色，重视团队合作与创新，并积极探索提高个人及团队效率的方法，同时关注工作与生活的平衡和个人职业发展。"
- 家庭方面："用户重视与家人的沟通，通过举办家庭活动如美食晚宴、郊游以及礼物交换增进情感联系。"
近期情境：
- "用户最近关注了如何在繁忙的生活中平衡家庭与工作的话题，浏览了一篇关于此主题的生活方式博客文章。"
"""

## 用户历史相关需求
以下列举了一些交互历史中用户在与当前相似的情境下曾经出现过的需求，包括总体的需求类型以及具体的需求内容。
"""
总体需求类型：
- "寻求改善工作与生活平衡的方法"
具体需求内容：
- "解决团队沟通效率低下的问题，特别是会议时间过长导致实际工作被耽误。"
- "希望在忙碌的工作之余与家人保持良好关系"
"""

# 当前用户询问
"""
如何更好地协调跨部门任务分工？
"""

# 输出模板
以下为json格式的输出模板，"requirement"为待生成的回复内容。
"""
{"requirement": "..."}
"""

# 输出要求
1. 结合输入中给出的用户历史信息，理解与当前用户询问相关的用户背景情境，用一句话的形式描述用户的实际需求。
2. 除要求的json格式输出外，不要输出任何其它内容，也不需要在json中添加注释进行额外说明。

现在，请你按照输出要求和模板给出输出。
````

效果：
````
{"requirement": "用户希望找到有效的方法或工具来优化跨部门之间的任务分配和协作流程，以提高工作效率。"}
````

reference：
````
"requirement": "用户希望通过引入高效的任务分配工具或方法，并结合提升沟通技巧，进一步优化跨部门协作流程，减少重复劳动，提高整体工作效率。"
````
