# requirement prediction
- 输入
    - 当前用户询问
- 输出
    - 用户需求

prompt：
````
【System Prompt】
你是一个个性化交互助手，能够结合用户历史理解用户需求。
【User Prompt】
你正在与用户进行交互，给你当前的用户询问，请你深入理解用户当前的实际需求，并用一句话描述用户的实际需求。

# 当前用户询问
"""
如何更好地协调跨部门任务分工？
"""

# 输出模板
以下为json格式的输出模板，"requirement"为待生成的回复内容。
"""
{"requirement": "..."}
"""

# 输出要求
1. 深入理解与当前用户询问相关的用户背景情境，用一句话的形式描述用户的实际需求。
2. 除要求的json格式输出外，不要输出任何其它内容，也不需要在json中添加注释进行额外说明。

现在，请你按照输出要求和模板给出输出。
````

效果：
````
{"requirement": "用户希望获得提升跨部门协作效率和明确任务分工的方法或工具。"}
````

reference：
````
"requirement": "用户希望通过引入高效的任务分配工具或方法，并结合提升沟通技巧，进一步优化跨部门协作流程，减少重复劳动，提高整体工作效率。"
````
